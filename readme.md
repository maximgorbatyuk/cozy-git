# Cozy Git

## Description

Cozy Git is a macOS application that provides a simple and intuitive interface for working with Git repositories. Key feature is focus on cleaning up git history and making it more readable, and also provide a way to automate some routine git operations.

## Key features

- CGF-1. Clean up git history. Merged branches will be deleted by user trigger. Also, there is a way to delete stale branches and remove them as well.
- CGF-2. Automate some routine git operations: put prefix to your commit messages, call any scripts on commit, push, pull, etc.
- Simple and intuitive interface
- macOS application

## UI guideline

The UI interface is clean and meanwhile Warm and Cozy. There are rounded corners, soft shadows, and warm colors. Avoid sharp corners, harsh shadows, and cold colors. Use SF Symbols for icons. Use system fonts. Use native macOS controls.
